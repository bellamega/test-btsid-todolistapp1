# To-Do List Application

Aplikasi To-Do List dengan fitur authentication dan checklist management.

## Features

- ✅ User Registration & Login
- ✅ JWT Authentication
- ✅ Create, Read, Update, Delete Checklists
- ✅ Manage Checklist Items
- ✅ Toggle Item Status
- ✅ Rename Items
- ✅ Responsive Design

## Tech Stack

- **Frontend**: React.js
- **Authentication**: JWT Token
- **API**: REST API
- **Styling**: Inline CSS
